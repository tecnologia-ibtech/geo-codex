# geo-pedidos — Knowledge Base para Força de Vendas (ISF)

Base de conhecimento arquitetural para o sistema **Força de Vendas** (ISF - Integração de Força de Vendas), uma aplicação offline-first para representantes comerciais no setor têxtil/confecção.

## O que cobre

- Arquitetura frontend (AngularJS 1.x + CoffeeScript + PouchDB)
- Arquitetura backend (Java REST + JPA/Hibernate + PostgreSQL)
- Padrão de sincronização offline-first
- Padrões de código (PersistService, Resource REST, Models, Entities)
- Endpoints principais e mapeamento frontend↔backend
- Guias para implementação de features
- Conceitos do domínio de negócio

## Para quem

Desenvolvedores trabalhando em:
- Repositório `pedidos` (frontend ISF)
- Repositório `geovendas-vaadin6` (backend)
- Qualquer funcionalidade relacionada ao Força de Vendas

## Comandos

- `/pedidos`: Carregar knowledge base do Força de Vendas (ISF)

## Instalação

```bash
claude install geo-pedidos
```
