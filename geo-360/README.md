# geo-360 Plugin

Base de conhecimento arquitetural para o ecossistema **CRM360** com 11 projetos integrados em 5 stacks tecnológicos.

## O que está incluído

- **11 Projetos**: Backend Java (4 libs + 2 apps), Frontends (Vue.js 2, Vaadin 6/23), APIs de Comunicação (.NET, Node.js)
- **5 Stacks**: Java 17, .NET 8, Node.js 20, Vue.js 2, Vaadin 6/23
- **Integrações**: WhatsApp (RabbitMQ), Webhooks, WebSocket real-time, Multi-tenancy
- **Padrões Arquiteturais**: 3-Layer, Repository, DI, Event-Driven, Multi-provider

## Quando instalar

Para desenvolvedores trabalhando com:
- geovendas-vue
- ibtech-geovendas / geovendas-vaadin23
- geochat-api
- Comm (.NET)
- Qualquer componente do workspace CRM360

## Conteúdo Principal

- **Visão Geral**: Arquitetura de alto nível, fluxos de dados, 11 projetos
- **Backend Java**: Entity, Repository, Service, Utils, Libraries
- **Frontends**: Vue.js SPA, Vaadin 6/23 (legacy + novo padrão)
- **Serviços de Comunicação**: geochat-api (Node.js), Comm (.NET)
- **Integrações**: RabbitMQ, HTTP Webhooks, WebSocket
- **Troubleshooting**: Checklists por stack
